The **Recipe Vault** is a relational digital cookbook platform designed to modernize legacy Meal-Master (.mmf) files. It features an advanced Python-based importer that utilizes regular expressions and heuristic analysis to "deconstruct" flat text into a structured SQLite database. Unlike standard converters, this system intelligently separates ingredient data into four distinct categories—quantity, unit, core item, and preparation/options—enabling high-level filtering and a cleaner user interface.

Built with a Flask backend and a Bootstrap 5 frontend, the application is optimized for network-wide accessibility. This allows the cookbook to be hosted on a central computer while remaining fully responsive and searchable on mobile devices or tablets used in the kitchen. With a high-contrast dark theme, integrated pagination, and automated cleanup of legacy version strings, it transforms thousands of disorganized text blocks into a professional, high-performance culinary library.
